Principais comandos do Git | sem Flow

Comando Descrição
$ git config –global user.name “Seu nome” O comando git config faz a configuração do usuário na máquina, ao adidionar a flag -global as configurações são aplicadas de forma global. Nesse exemplo configurando o nome do usuário
$ git config –global user.email “Seu e-mail” O comando git config faz a configuração do usuário na máquina, ao adidionar a flag -global as configurações são aplicadas de forma global. Nesse exemplo configurando o e-mail do usuário
